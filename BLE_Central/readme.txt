Some useful BLE GAP Central example projects.

Minimal_Peripheral.cydsn:
GATT Server on GAP Peripheral project used by the GAP Central examples
Runs on CY5670, CySmart USB Dongle

Mnimal.cydsn:
Minimalistic code example of a GATT Client on a GAP Central

AutoDiscovery.cydsn:
GATT Client performing automatic discovery using CyBle_GattcStartDiscovery() API

All projects tested using CY8CKIT-042-BLE kit.
